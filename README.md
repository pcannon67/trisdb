trisdb
======

An open-source in-memory triplestore database with a beautiful GUI and incredible performance.

### Requirements
* CMake >= v2.8
* C++11 compatible compiler (e.g. g++ 4.8)

### Dependencies
* libboost-all-dev >= 1.48
* libreadline6-dev
* libprotobuf-dev
* libsnappy

### Installation
```
cd build/
cmake ../src/
make
sudo make install
```

### Usage
Start TrisDb
```
sudo start trisdb
```
Open the shell
```
trisdb --shell
```
Stop TrisDb
```
sudo stop trisdb
```
